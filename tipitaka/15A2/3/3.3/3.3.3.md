# 3.3.3

### Tatiyapaṇṇāsaka

### Dānavagga

(144.)

235\. “Dveme, bhikkhave, cāgā. Katame dve? Āmisacāgo ca dhammacāgo ca. Ime kho, bhikkhave, dve cāgā. Etadaggaṃ, bhikkhave, imesaṃ dvinnaṃ cāgānaṃ yadidaṃ dhammacāgo”ti.
