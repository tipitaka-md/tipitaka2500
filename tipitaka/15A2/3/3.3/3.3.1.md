# 3.3.1

### Tatiyapaṇṇāsaka

### Dānavagga

(142.)

233. “Dvemāni, bhikkhave, dānāni. Katamāni dve? Āmisadānañca dhammadānañca. Imāni kho, bhikkhave, dve dānāni. Etadaggaṃ, bhikkhave, imesaṃ dvinnaṃ dānānaṃ yadidaṃ dhammadānan”ti.
