# 3.2.1

### Tatiyapaṇṇāsaka

### Āyācanavagga

(131.)

217. “Saddho, bhikkhave, bhikkhu evaṃ sammā āyācamāno āyāceyya—  ‘tādiso homi yādisā sāriputtamoggallānā’ti. Esā, bhikkhave, tulā etaṃ pamāṇaṃ mama sāvakānaṃ bhikkhūnaṃ yadidaṃ sāriputtamoggallānā”ti.
