# 3.3.3 Tatiyasutta

### Tatiyapaṇṇāsaka

### Parisuddhavagga

### Tatiyasutta

(125.)

1086\. “Dasayime, bhikkhave, dhammā mahapphalā mahānisaṃsā, nāññatra sugatavinayā. Katame dasa? Sammādiṭṭhi…pe…  sammāvimutti—  ime kho, bhikkhave, dasa dhammā mahapphalā mahānisaṃsā, nāññatra sugatavinayā”ti.

---

1087\. Tatiyaṃ.
