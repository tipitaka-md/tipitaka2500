# 3.3.2 Dutiyasutta

### Tatiyapaṇṇāsaka

### Parisuddhavagga

### Dutiyasutta

(124.)

1084. “Dasayime, bhikkhave, dhammā anuppannā uppajjanti, nāññatra sugatavinayā. Katame dasa? Sammādiṭṭhi…pe…  sammāvimutti—  ime kho, bhikkhave, dasa dhammā anuppannā uppajjanti, nāññatra sugatavinayā”ti.

===

1085. Dutiyaṃ.
