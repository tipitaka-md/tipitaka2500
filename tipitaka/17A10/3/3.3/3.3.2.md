1084\. “𑀤𑀲𑀬𑀺𑀫𑁂, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀥𑀫𑁆𑀫𑀸 𑀅𑀦𑀼𑀧𑁆𑀧𑀦𑁆𑀦𑀸 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀦𑁆𑀢𑀺, 𑀦𑀸𑀜𑁆𑀜𑀢𑁆𑀭 𑀲𑀼𑀕𑀢𑀯𑀺𑀦𑀬𑀸𑁇 𑀓𑀢𑀫𑁂 𑀤𑀲? 𑀲𑀫𑁆𑀫𑀸𑀤𑀺𑀝𑁆𑀞𑀺…𑀧𑁂…  𑀲𑀫𑁆𑀫𑀸𑀯𑀺𑀫𑀼𑀢𑁆𑀢𑀺—  𑀇𑀫𑁂 𑀔𑁄, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀤𑀲 𑀥𑀫𑁆𑀫𑀸 𑀅𑀦𑀼𑀧𑁆𑀧𑀦𑁆𑀦𑀸 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀦𑁆𑀢𑀺, 𑀦𑀸𑀜𑁆𑀜𑀢𑁆𑀭 𑀲𑀼𑀕𑀢𑀯𑀺𑀦𑀬𑀸”𑀢𑀺𑁇
