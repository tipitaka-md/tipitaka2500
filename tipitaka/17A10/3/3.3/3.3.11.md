# 𑁩𑁇𑁩𑁇𑁧𑁧 𑀏𑀓𑀸𑀤𑀲𑀫𑀲𑀼𑀢𑁆𑀢

### 𑀢𑀢𑀺𑀬𑀧𑀡𑁆𑀡𑀸𑀲𑀓

### 𑀧𑀭𑀺𑀲𑀼𑀤𑁆𑀥𑀯𑀕𑁆𑀕

### 𑀏𑀓𑀸𑀤𑀲𑀫𑀲𑀼𑀢𑁆𑀢

(133.)

1102. “𑀤𑀲𑀬𑀺𑀫𑁂, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀸𑁇 𑀓𑀢𑀫𑁂 𑀤𑀲? 𑀲𑀫𑁆𑀫𑀸𑀤𑀺𑀝𑁆𑀞𑀺, 𑀲𑀫𑁆𑀫𑀸𑀲𑀗𑁆𑀓𑀧𑁆𑀧𑁄, 𑀲𑀫𑁆𑀫𑀸𑀯𑀸𑀘𑀸, 𑀲𑀫𑁆𑀫𑀸𑀓𑀫𑁆𑀫𑀦𑁆𑀢𑁄, 𑀲𑀫𑁆𑀫𑀸𑀆𑀚𑀻𑀯𑁄, 𑀲𑀫𑁆𑀫𑀸𑀯𑀸𑀬𑀸𑀫𑁄, 𑀲𑀫𑁆𑀫𑀸𑀲𑀢𑀺, 𑀲𑀫𑁆𑀫𑀸𑀲𑀫𑀸𑀥𑀺, 𑀲𑀫𑁆𑀫𑀸𑀜𑀸𑀡𑀁, 𑀲𑀫𑁆𑀫𑀸𑀯𑀺𑀫𑀼𑀢𑁆𑀢𑀺—  𑀇𑀫𑁂 𑀔𑁄, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀤𑀲 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀸”𑀢𑀺𑁇
