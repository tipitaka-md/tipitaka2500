# 3 Kusalattika, Hetusampayuttaduka

### Kusalattika, Hetusampayuttaduka

(10.)

27. Kusalaṃ hetusampayuttaṃ dhammaṃ paṭicca nakusalo nahetusampayutto dhammo uppajjati hetupaccayā. (Sahetukadukasadisaṃ.)
