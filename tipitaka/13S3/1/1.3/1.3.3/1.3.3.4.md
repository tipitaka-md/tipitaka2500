# 1.3.3.4 Assādasutta

### Khandhasaṃyutta

### Uparipaṇṇāsaka

### Avijjāvagga

### Assādasutta

(129.)

731. Bārāṇasiyaṃ viharanti isipatane migadāye…pe…  ekamantaṃ nisinno kho āyasmā mahākoṭṭhiko āyasmantaṃ sāriputtaṃ etadavoca—  “‘avijjā, avijjā’ti, āvuso sāriputta, vuccati. Katamā nu kho, āvuso, avijjā; kittāvatā ca avijjāgato hotī”ti?

732. “Idhāvuso assutavā puthujjano rūpassa assādañca ādīnavañca nissaraṇañca yathābhūtaṃ nappajānāti. Vedanāya…pe…  saññāya…  saṅkhārānaṃ…  viññāṇassa assādañca ādīnavañca nissaraṇañca yathābhūtaṃ nappajānāti. Ayaṃ vuccatāvuso, avijjā; ettāvatā ca avijjāgato hotī”ti.
