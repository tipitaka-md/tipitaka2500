(14.)

1130. Kusalo anidassano dhammo kusalassa anidassanassa dhammassa hetupaccayena paccayo…  tīṇi.

1131. Akusalo anidassano dhammo akusalassa anidassanassa dhammassa hetupaccayena paccayo…  tīṇi.

1132. Abyākato anidassano dhammo abyākatassa anidassanassa dhammassa hetupaccayena paccayo. (1)

(15.)

1133. Kusalo anidassano dhammo kusalassa anidassanassa dhammassa ārammaṇapaccayena paccayo…  tīṇi.

1134. Akusalo anidassano dhammo akusalassa anidassanassa dhammassa ārammaṇapaccayena paccayo…  tīṇi.

1135. Abyākato anidassano dhammo abyākatassa anidassanassa dhammassa ārammaṇapaccayena paccayo…  tīṇi. (Saṃkhittaṃ.)

(16.)

1136. Hetuyā satta, ārammaṇe nava, adhipatiyā dasa, anantare satta…pe…  upanissaye nava…pe…  avigate terasa. (Saṃkhittaṃ.)

1137. (Yathā kusalattike pañhāvāraṃ, evaṃ vitthāretabbaṃ.)
