# 1.4.6 Dutiyamittasutta

### Paṭhamapaṇṇāsaka

### Devatāvagga

### Dutiyamittasutta

(37.)

197. “Sattahi, bhikkhave, dhammehi samannāgato bhikkhu mitto sevitabbo bhajitabbo payirupāsitabbo api panujjamānenapi. Katamehi sattahi? Piyo ca hoti manāpo ca garu ca bhāvanīyo ca vattā ca vacanakkhamo ca gambhīrañca kathaṃ kattā hoti, no ca aṭṭhāne niyojeti. Imehi kho, bhikkhave, sattahi dhammehi samannāgato bhikkhu mitto sevitabbo bhajitabbo payirupāsitabbo api panujjamānenapīti.

198. _Piyo garu bhāvanīyo,_  
_vattā ca vacanakkhamo;_  
_Gambhīrañca kathaṃ kattā,_  
_no caṭṭhāne niyojako._  


199. _Yamhi etāni ṭhānāni,_  
_saṃvijjantīdha puggale;_  
_So mitto mittakāmena,_  
_atthakāmānukampato;_  
_Api nāsiyamānena,_  
_bhajitabbo tathāvidho”ti._  


===

200. Chaṭṭhaṃ.
