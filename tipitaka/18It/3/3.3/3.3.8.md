# 3.3.8 Bhidurasutta

### Tikanipāta

### Tatiyavagga

### Bhidurasutta

(77.)

510. Vuttañhetaṃ bhagavatā vuttamarahatāti me sutaṃ—

511. “Bhidurāyaṃ, bhikkhave, kāyo, viññāṇaṃ virāgadhammaṃ, sabbe upadhī aniccā dukkhā vipariṇāmadhammā”ti. Etamatthaṃ bhagavā avoca. Tatthetaṃ iti vuccati—

512. _“Kāyañca bhiduraṃ ñatvā,_  
_Viññāṇañca virāgunaṃ;_  
_Upadhīsu bhayaṃ disvā,_  
_Jātimaraṇamaccagā;_  
_Sampatvā paramaṃ santiṃ,_  
_Kālaṃ kaṅkhati bhāvitatto”ti._  


513. Ayampi attho vutto bhagavatā, iti me sutanti.

===

514. Aṭṭhamaṃ.
