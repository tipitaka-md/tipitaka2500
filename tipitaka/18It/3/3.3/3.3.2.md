# 3.3.2 Sammādiṭṭhikasutta

* Tikanipāta

* Tatiyavagga

* Sammādiṭṭhikasutta

(71.)

457\. Vuttañhetaṃ bhagavatā vuttamarahatāti me sutaṃ—

458\. “Diṭṭhā mayā, bhikkhave, sattā kāyasucaritena samannāgatā vacīsucaritena samannāgatā manosucaritena samannāgatā ariyānaṃ anupavādakā sammādiṭṭhikā sammādiṭṭhikammasamādānā. Te kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapannā.

459\. Taṃ kho panāhaṃ, bhikkhave, nāññassa samaṇassa vā brāhmaṇassa vā sutvā vadāmi. Diṭṭhā mayā, bhikkhave, sattā kāyasucaritena samannāgatā vacīsucaritena samannāgatā manosucaritena samannāgatā ariyānaṃ anupavādakā sammādiṭṭhikā sammādiṭṭhikammasamādānā. Te kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapannā. Api ca, bhikkhave, yadeva sāmaṃ ñātaṃ sāmaṃ diṭṭhaṃ sāmaṃ viditaṃ tadevāhaṃ vadāmi.

460\. Diṭṭhā mayā, bhikkhave, sattā kāyasucaritena samannāgatā vacīsucaritena samannāgatā manosucaritena samannāgatā ariyānaṃ anupavādakā sammādiṭṭhikā sammādiṭṭhikammasamādānā. Te kāyassa bhedā paraṃ maraṇā sugatiṃ saggaṃ lokaṃ upapannā”ti. Etamatthaṃ bhagavā avoca. Tatthetaṃ iti vuccati—

461\. _“Sammā manaṃ paṇidhāya,_  
_sammā vācañca bhāsiya;_  
_Sammā kammāni katvāna,_  
_kāyena idha puggalo._  


462\. _Bahussuto puññakaro,_  
_appasmiṃ idha jīvite;_  
_Kāyassa bhedā sappañño,_  
_saggaṃ so upapajjatī”ti._  


463\. Ayampi attho vutto bhagavatā, iti me sutanti.

---

464\. Dutiyaṃ.
