# 2.3 Cūḷapanthakavatthu

* Appamādavagga

* Cūḷapanthakavatthu

(25.)

27\. _Uṭṭhānenappamādena,_  
_saṃyamena damena ca;_  
_Dīpaṃ kayirātha medhāvī,_  
_yaṃ ogho nābhikīrati._
