# 3.2 Aññatarabhikkhuvatthu

### Cittavagga

### Aññatarabhikkhuvatthu

(35.)

38. _Dunniggahassa lahuno,_  
_yatthakāmanipātino;_  
_Cittassa damatho sādhu,_  
_cittaṃ dantaṃ sukhāvahaṃ._
