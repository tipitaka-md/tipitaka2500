# 3.4 Saṃgharakkhitabhāgineyyattheravatthu

* Cittavagga

* Saṃgharakkhitabhāgineyyattheravatthu

(37.)

40\. _Dūraṅgamaṃ ekacaraṃ,_  
_asarīraṃ guhāsayaṃ;_  
_Ye cittaṃ saṃyamissanti,_  
_mokkhanti mārabandhanā._
