# 3.3 Aññataraukkaṇṭhitabhikkhuvatthu

### Cittavagga

### Aññataraukkaṇṭhitabhikkhuvatthu

(36.)

39. _Sududdasaṃ sunipuṇaṃ,_  
_yatthakāmanipātinaṃ;_  
_Cittaṃ rakkhetha medhāvī,_  
_cittaṃ guttaṃ sukhāvahaṃ._
