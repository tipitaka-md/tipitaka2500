# 3.7 Pūtigattatissattheravatthu

* Cittavagga

* Pūtigattatissattheravatthu

(41.)

44\. _Aciraṃ vatayaṃ kāyo,_  
_pathaviṃ adhisessati;_  
_Chuddho apetaviññāṇo,_  
_niratthaṃva kaliṅgaraṃ._
