# 𑁩𑁇𑁧 𑀫𑁂𑀖𑀺𑀬𑀢𑁆𑀣𑁂𑀭𑀯𑀢𑁆𑀣𑀼

### 𑀘𑀺𑀢𑁆𑀢𑀯𑀕𑁆𑀕

### 𑀫𑁂𑀖𑀺𑀬𑀢𑁆𑀣𑁂𑀭𑀯𑀢𑁆𑀣𑀼

(33.)

36. _𑀨𑀦𑁆𑀤𑀦𑀁 𑀘𑀧𑀮𑀁 𑀘𑀺𑀢𑁆𑀢𑀁,_  
_𑀤𑀽𑀭𑀓𑁆𑀔𑀁 𑀤𑀼𑀦𑁆𑀦𑀺𑀯𑀸𑀭𑀬𑀁;_  
_𑀉𑀚𑀼𑀁 𑀓𑀭𑁄𑀢𑀺 𑀫𑁂𑀥𑀸𑀯𑀻,_  
_𑀉𑀲𑀼𑀓𑀸𑀭𑁄𑀯 𑀢𑁂𑀚𑀦𑀁𑁇_  


(34.)

37. _𑀯𑀸𑀭𑀺𑀚𑁄𑀯 𑀣𑀮𑁂 𑀔𑀺𑀢𑁆𑀢𑁄,_  
_𑀑𑀓𑀫𑁄𑀓𑀢𑁅𑀩𑁆𑀪𑀢𑁄;_  
_𑀧𑀭𑀺𑀨𑀦𑁆𑀤𑀢𑀺𑀤𑀁 𑀘𑀺𑀢𑁆𑀢𑀁,_  
_𑀫𑀸𑀭𑀥𑁂𑀬𑁆𑀬𑀁 𑀧𑀳𑀸𑀢𑀯𑁂𑁇_
