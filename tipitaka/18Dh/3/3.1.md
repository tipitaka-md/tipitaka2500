# 3.1 Meghiyattheravatthu

* Cittavagga

* Meghiyattheravatthu

(33.)

36\. _Phandanaṃ capalaṃ cittaṃ,_  
_dūrakkhaṃ dunnivārayaṃ;_  
_Ujuṃ karoti medhāvī,_  
_usukārova tejanaṃ._  


(34.)

37\. _Vārijova thale khitto,_  
_okamokataubbhato;_  
_Pariphandatidaṃ cittaṃ,_  
_māradheyyaṃ pahātave._
