# 14.7 Ānandattherapañhavatthu

* Buddhavagga

* Ānandattherapañhavatthu

(193.)

207\. _Dullabho purisājañño,_  
_na so sabbattha jāyati;_  
_Yattha so jāyati dhīro,_  
_taṃ kulaṃ sukhamedhati._
