# 2.3.2.2.2 Saṅkhataduka

* Niddesa

* Nikkhepakaṇḍa

* Dukanikkhepa

* Cūḷantaraduka

* Saṅkhataduka

(1091.)

1760\. Katame dhammā saṅkhatā? Yeva te dhammā sappaccayā, teva te dhammā saṅkhatā.

(1092.)

1761\. Katame dhammā asaṅkhatā? Yo eva so dhammo appaccayo, so eva so dhammo asaṅkhato.
