# 2.3.3.36 Sīlasampadāduka

* Niddesa

* Nikkhepakaṇḍa

* Suttantikadukanikkhepa

* Sīlasampadāduka

(1370.)

2075\. Tattha katamā sīlasampadā? Yo kāyiko avītikkamo, vācasiko avītikkamo, kāyikavācasiko avītikkamo—  ayaṃ vuccati sīlasampadā. Sabbopi sīlasaṃvaro sīlasampadā.

(1371.)

2076\. Tattha katamā diṭṭhisampadā? Atthi dinnaṃ, atthi yiṭṭhaṃ, atthi hutaṃ, atthi sukatadukkaṭānaṃ kammānaṃ phalaṃ vipāko, atthi ayaṃ loko, atthi paro loko, atthi mātā, atthi pitā, atthi sattā opapātikā, atthi loke samaṇabrāhmaṇā sammaggatā sammāpaṭipannā ye imañca lokaṃ parañca lokaṃ sayaṃ abhiññā sacchikatvā pavedentīti; yā evarūpā paññā pajānanā…pe…  amoho dhammavicayo sammādiṭṭhi—  ayaṃ vuccati diṭṭhisampadā. Sabbāpi sammādiṭṭhi diṭṭhisampadā.
