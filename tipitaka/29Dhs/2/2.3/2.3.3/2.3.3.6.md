# 2.3.3.6 Adhivacanaduka

### Niddesa

### Nikkhepakaṇḍa

### Suttantikadukanikkhepa

### Adhivacanaduka

(1313.)

2015. Katame dhammā adhivacanā? Yā tesaṃ tesaṃ dhammānaṃ saṅkhā samaññā paññatti vohāro nāmaṃ nāmakammaṃ nāmadheyyaṃ nirutti byañjanaṃ abhilāpo—  ime dhammā adhivacanā.

2016. Sabbeva dhammā adhivacanapathā.
