# 2.3.3.13 Antavādiṭṭhiduka

### Niddesa

### Nikkhepakaṇḍa

### Suttantikadukanikkhepa

### Antavādiṭṭhiduka

(1324.)

2029. Tattha katamā antavā diṭṭhi? Antavā attā ca loko cāti, yā evarūpā diṭṭhi diṭṭhigataṃ…pe…  vipariyāsaggāho—  ayaṃ vuccati antavā diṭṭhi.

(1325.)

2030. Tattha katamā anantavā diṭṭhi? Anantavā attā ca loko cāti, yā evarūpā diṭṭhi diṭṭhigataṃ…pe…  vipariyāsaggāho—  ayaṃ vuccati anantavā diṭṭhi.
