# 2.3.3.8 Paññattiduka

### Niddesa

### Nikkhepakaṇḍa

### Suttantikadukanikkhepa

### Paññattiduka

(1315.)

2019. Katame dhammā paññatti? Yā tesaṃ tesaṃ dhammānaṃ saṅkhā samaññā paññatti vohāro nāmaṃ nāmakammaṃ nāmadheyyaṃ nirutti byañjanaṃ abhilāpo—  ime dhammā paññatti.

2020. Sabbeva dhammā paññattipathā.
