# 2.3.3.37 Sīlavisuddhiduka

* Niddesa

* Nikkhepakaṇḍa

* Suttantikadukanikkhepa

* Sīlavisuddhiduka

(1372.)

2077\. Tattha katamā sīlavisuddhi? Yo kāyiko avītikkamo, vācasiko avītikkamo, kāyikavācasiko avītikkamo—  ayaṃ vuccati sīlavisuddhi. Sabbopi sīlasaṃvaro sīlavisuddhi.

(1373.)

2078\. Tattha katamā diṭṭhivisuddhi? Kammassakatañāṇaṃ saccānulomikañāṇaṃ maggasamaṅgissa ñāṇaṃ phalasamaṅgissa ñāṇaṃ.
