# 2.3.3.16 Hiriduka

### Niddesa

### Nikkhepakaṇḍa

### Suttantikadukanikkhepa

### Hiriduka

(1330.)

2035. Tattha katamā hirī? Yaṃ hirīyati hiriyitabbena, hirīyati pāpakānaṃ akusalānaṃ dhammānaṃ samāpattiyā—  ayaṃ vuccati hirī.

(1331.)

2036. Tattha katamaṃ ottappaṃ? Yaṃ ottappati ottappitabbena, ottappati pāpakānaṃ akusalānaṃ dhammānaṃ samāpattiyā—  idaṃ vuccati ottappaṃ.
