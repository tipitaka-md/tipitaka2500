# 𑁨𑁇𑁩𑁇𑁩𑁇𑁨𑁯 𑀫𑀼𑀝𑁆𑀞𑀲𑀘𑁆𑀘𑀤𑀼𑀓

### 𑀦𑀺𑀤𑁆𑀤𑁂𑀲

### 𑀦𑀺𑀓𑁆𑀔𑁂𑀧𑀓𑀡𑁆𑀟

### 𑀲𑀼𑀢𑁆𑀢𑀦𑁆𑀢𑀺𑀓𑀤𑀼𑀓𑀦𑀺𑀓𑁆𑀔𑁂𑀧

### 𑀫𑀼𑀝𑁆𑀞𑀲𑀘𑁆𑀘𑀤𑀼𑀓

(1356.)

2061\. 𑀢𑀢𑁆𑀣 𑀓𑀢𑀫𑀁 𑀫𑀼𑀝𑁆𑀞𑀲𑀘𑁆𑀘𑀁? 𑀬𑀸 𑀅𑀲𑀢𑀺 𑀅𑀦𑀦𑀼𑀲𑁆𑀲𑀢𑀺 𑀅𑀧𑁆𑀧𑀝𑀺𑀲𑁆𑀲𑀢𑀺 𑀅𑀲𑀢𑀺 𑀅𑀲𑀭𑀡𑀢𑀸 𑀅𑀥𑀸𑀭𑀡𑀢𑀸 𑀧𑀺𑀮𑀸𑀧𑀦𑀢𑀸 𑀲𑀫𑁆𑀫𑀼𑀲𑁆𑀲𑀦𑀢𑀸—  𑀇𑀤𑀁 𑀯𑀼𑀘𑁆𑀘𑀢𑀺 𑀫𑀼𑀝𑁆𑀞𑀲𑀘𑁆𑀘𑀁𑁇

(1357.)

2062\. 𑀢𑀢𑁆𑀣 𑀓𑀢𑀫𑀁 𑀅𑀲𑀫𑁆𑀧𑀚𑀜𑁆𑀜𑀁? 𑀬𑀁 𑀅𑀜𑁆𑀜𑀸𑀡𑀁 𑀅𑀤𑀲𑁆𑀲𑀦𑀁…𑀧𑁂…  𑀅𑀯𑀺𑀚𑁆𑀚𑀸𑀮𑀗𑁆𑀕𑀻 𑀫𑁄𑀳𑁄 𑀅𑀓𑀼𑀲𑀮𑀫𑀽𑀮𑀁—  𑀇𑀤𑀁 𑀯𑀼𑀘𑁆𑀘𑀢𑀺 𑀅𑀲𑀫𑁆𑀧𑀚𑀜𑁆𑀜𑀁𑁇
