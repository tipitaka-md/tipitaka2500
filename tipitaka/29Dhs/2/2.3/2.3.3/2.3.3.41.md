# 2.3.3.41 Vijjāduka

### Niddesa

### Nikkhepakaṇḍa

### Suttantikadukanikkhepa

### Vijjāduka

(1380.)

2085. **Vijjā**ti—  tisso vijjā—  pubbenivāsānussati ñāṇaṃ vijjā, sattānaṃ cutūpapāte ñāṇaṃ vijjā, āsavānaṃ khaye ñāṇaṃ vijjā.

(1381.)

2086. **Vimuttī**ti—  dve vimuttiyo—  cittassa adhimutti, nibbānañca.
