# 2.3.3.7 Niruttiduka

* Niddesa

* Nikkhepakaṇḍa

* Suttantikadukanikkhepa

* Niruttiduka

(1314.)

2017\. Katame dhammā nirutti? Yā tesaṃ tesaṃ dhammānaṃ saṅkhā samaññā paññatti vohāro nāmaṃ nāmakammaṃ nāmadheyyaṃ nirutti byañjanaṃ abhilāpo—  ime dhammā nirutti.

2018\. Sabbeva dhammā niruttipathā.
