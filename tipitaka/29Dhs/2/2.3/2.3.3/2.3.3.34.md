# 2.3.3.34 Paggāhaduka

* Niddesa

* Nikkhepakaṇḍa

* Suttantikadukanikkhepa

* Paggāhaduka

(1366.)

2071\. Tattha katamo paggāho? Yo cetasiko vīriyārambho…pe…  sammāvāyāmo—  ayaṃ vuccati paggāho.

(1367.)

2072\. Tattha katamo avikkhepo? Yā cittassa ṭhiti…pe…  sammāsamādhi—  ayaṃ vuccati avikkhepo.
