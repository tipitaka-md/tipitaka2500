# 𑁫𑁪𑁇𑁫 𑀓𑀼𑀫𑀸𑀭𑀓𑀲𑁆𑀲𑀧𑀢𑁆𑀣𑁂𑀭𑀅𑀧𑀤𑀸𑀦

* 𑀓𑀘𑁆𑀘𑀸𑀬𑀦𑀯𑀕𑁆𑀕

* 𑀓𑀼𑀫𑀸𑀭𑀓𑀲𑁆𑀲𑀧𑀢𑁆𑀣𑁂𑀭𑀅𑀧𑀤𑀸𑀦

(150.)

7105\. _“𑀇𑀢𑁄 𑀲𑀢𑀲𑀳𑀲𑁆𑀲𑀫𑁆𑀳𑀺,_  
_𑀓𑀧𑁆𑀧𑁂 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀺 𑀦𑀸𑀬𑀓𑁄;_  
_𑀲𑀩𑁆𑀩𑀮𑁄𑀓𑀳𑀺𑀢𑁄 𑀯𑀻𑀭𑁄,_  
_𑀧𑀤𑀼𑀫𑀼𑀢𑁆𑀢𑀭𑀦𑀸𑀫𑀓𑁄𑁇_  


(151.)

7106\. _𑀢𑀤𑀸𑀳𑀁 𑀩𑁆𑀭𑀸𑀳𑁆𑀫𑀡𑁄 𑀳𑀼𑀢𑁆𑀯𑀸,_  
_𑀯𑀺𑀲𑁆𑀲𑀼𑀢𑁄 𑀯𑁂𑀤𑀧𑀸𑀭𑀕𑀽;_  
_𑀤𑀺𑀯𑀸𑀯𑀺𑀳𑀸𑀭𑀁 𑀯𑀺𑀘𑀭𑀁,_  
_𑀅𑀤𑁆𑀤𑀲𑀁 𑀮𑁄𑀓𑀦𑀸𑀬𑀓𑀁𑁇_  


(152.)

7107\. _𑀘𑀢𑀼𑀲𑀘𑁆𑀘𑀁 𑀧𑀓𑀸𑀲𑁂𑀦𑁆𑀢𑀁,_  
_𑀩𑁄𑀥𑀬𑀦𑁆𑀢𑀁 𑀲𑀤𑁂𑀯𑀓𑀁;_  
_𑀯𑀺𑀘𑀺𑀢𑁆𑀢𑀓𑀣𑀺𑀓𑀸𑀦𑀕𑁆𑀕𑀁,_  
_𑀯𑀡𑁆𑀡𑀬𑀦𑁆𑀢𑀁 𑀫𑀳𑀸𑀚𑀦𑁂𑁇_  


(153.)

7108\. _𑀢𑀤𑀸 𑀫𑀼𑀤𑀺𑀢𑀘𑀺𑀢𑁆𑀢𑁄𑀳𑀁,_  
_𑀦𑀺𑀫𑀦𑁆𑀢𑁂𑀢𑁆𑀯𑀸 𑀢𑀣𑀸𑀕𑀢𑀁;_  
_𑀦𑀸𑀦𑀸𑀭𑀢𑁆𑀢𑁂𑀳𑀺 𑀯𑀢𑁆𑀣𑁂𑀳𑀺,_  
_𑀅𑀮𑀗𑁆𑀓𑀭𑀺𑀢𑁆𑀯𑀸𑀦 𑀫𑀡𑁆𑀟𑀧𑀁𑁇_  


(154.)

7109\. _𑀦𑀸𑀦𑀸𑀭𑀢𑀦𑀧𑀚𑁆𑀚𑁄𑀢𑀁,_  
_𑀲𑀲𑀁𑀖𑀁 𑀪𑁄𑀚𑀬𑀺𑀁 𑀢𑀳𑀺𑀁;_  
_𑀪𑁄𑀚𑀬𑀺𑀢𑁆𑀯𑀸𑀦 𑀲𑀢𑁆𑀢𑀸𑀳𑀁,_  
_𑀦𑀸𑀦𑀕𑁆𑀕𑀭𑀲𑀪𑁄𑀚𑀦𑀁𑁇_  


(155.)

7110\. _𑀦𑀸𑀦𑀸𑀘𑀺𑀢𑁆𑀢𑁂𑀳𑀺 𑀧𑀼𑀧𑁆𑀨𑁂𑀳𑀺,_  
_𑀧𑀽𑀚𑀬𑀺𑀢𑁆𑀯𑀸 𑀲𑀲𑀸𑀯𑀓𑀁;_  
_𑀦𑀺𑀧𑀘𑁆𑀘 𑀧𑀸𑀤𑀫𑀽𑀮𑀫𑁆𑀳𑀺,_  
_𑀢𑀁 𑀞𑀸𑀦𑀁 𑀧𑀢𑁆𑀣𑀬𑀺𑀁 𑀅𑀳𑀁𑁇_  


(156.)

7111\. _𑀢𑀤𑀸 𑀫𑀼𑀦𑀺𑀯𑀭𑁄 𑀆𑀳,_  
_𑀓𑀭𑀼𑀡𑁂𑀓𑀭𑀲𑀸𑀲𑀬𑁄;_  
_‘𑀧𑀲𑁆𑀲𑀣𑁂𑀢𑀁 𑀤𑀺𑀚𑀯𑀭𑀁,_  
_𑀧𑀤𑀼𑀫𑀸𑀦𑀦𑀮𑁄𑀘𑀦𑀁𑁇_  


(157.)

7112\. _𑀧𑀻𑀢𑀺𑀧𑀸𑀫𑁄𑀚𑁆𑀚𑀩𑀳𑀼𑀮𑀁,_  
_𑀲𑀫𑀼𑀕𑁆𑀕𑀢𑀢𑀦𑀽𑀭𑀼𑀳𑀁;_  
_𑀳𑀸𑀲𑀫𑁆𑀳𑀺𑀢𑀯𑀺𑀲𑀸𑀮𑀓𑁆𑀔𑀁,_  
_𑀫𑀫 𑀲𑀸𑀲𑀦𑀮𑀸𑀮𑀲𑀁𑁇_  


(158.)

7113\. _𑀧𑀢𑀺𑀢𑀁 𑀧𑀸𑀤𑀫𑀽𑀮𑁂 𑀫𑁂,_  
_𑀏𑀓𑀸𑀯𑀢𑁆𑀣𑀲𑀼𑀫𑀸𑀦𑀲𑀁;_  
_𑀏𑀲 𑀧𑀢𑁆𑀣𑁂𑀢𑀺 𑀢𑀁 𑀞𑀸𑀦𑀁,_  
_𑀯𑀺𑀘𑀺𑀢𑁆𑀢𑀓𑀣𑀺𑀓𑀢𑁆𑀢𑀦𑀁𑁇_  


(159.)

7114\. _𑀲𑀢𑀲𑀳𑀲𑁆𑀲𑀺𑀢𑁄 𑀓𑀧𑁆𑀧𑁂,_  
_𑀑𑀓𑁆𑀓𑀸𑀓𑀓𑀼𑀮𑀲𑀫𑁆𑀪𑀯𑁄;_  
_𑀕𑁄𑀢𑀫𑁄 𑀦𑀸𑀫 𑀕𑁄𑀢𑁆𑀢𑁂𑀦,_  
_𑀲𑀢𑁆𑀣𑀸 𑀮𑁄𑀓𑁂 𑀪𑀯𑀺𑀲𑁆𑀲𑀢𑀺𑁇_  


(160.)

7115\. _𑀢𑀲𑁆𑀲 𑀥𑀫𑁆𑀫𑁂𑀲𑀼 𑀤𑀸𑀬𑀸𑀤𑁄,_  
_𑀑𑀭𑀲𑁄 𑀥𑀫𑁆𑀫𑀦𑀺𑀫𑁆𑀫𑀺𑀢𑁄;_  
_𑀓𑀼𑀫𑀸𑀭𑀓𑀲𑁆𑀲𑀧𑁄 𑀦𑀸𑀫,_  
_𑀳𑁂𑀲𑁆𑀲𑀢𑀺 𑀲𑀢𑁆𑀣𑀼 𑀲𑀸𑀯𑀓𑁄𑁇_  


(161.)

7116\. _𑀯𑀺𑀘𑀺𑀢𑁆𑀢𑀧𑀼𑀧𑁆𑀨𑀤𑀼𑀲𑁆𑀲𑀸𑀦𑀁,_  
_𑀭𑀢𑀦𑀸𑀦𑀜𑁆𑀘 𑀯𑀸𑀳𑀲𑀸;_  
_𑀯𑀺𑀘𑀺𑀢𑁆𑀢𑀓𑀣𑀺𑀓𑀸𑀦𑀁 𑀲𑁄,_  
_𑀅𑀕𑁆𑀕𑀢𑀁 𑀧𑀸𑀧𑀼𑀡𑀺𑀲𑁆𑀲𑀢𑀺’𑁇_  


(162.)

7117\. _𑀢𑁂𑀦 𑀓𑀫𑁆𑀫𑁂𑀦 𑀲𑀼𑀓𑀢𑁂𑀦,_  
_𑀘𑁂𑀢𑀦𑀸𑀧𑀡𑀺𑀥𑀻𑀳𑀺 𑀘;_  
_𑀚𑀳𑀺𑀢𑁆𑀯𑀸 𑀫𑀸𑀦𑀼𑀲𑀁 𑀤𑁂𑀳𑀁,_  
_𑀢𑀸𑀯𑀢𑀺𑀁𑀲𑀫𑀕𑀘𑁆𑀙𑀳𑀁𑁇_  


(163.)

7118\. _𑀧𑀭𑀺𑀩𑁆𑀪𑀫𑀁 𑀪𑀯𑀸𑀪𑀯𑁂,_  
_𑀭𑀗𑁆𑀕𑀫𑀚𑁆𑀛𑁂 𑀬𑀣𑀸 𑀦𑀝𑁄;_  
_𑀲𑀸𑀔𑀫𑀺𑀕𑀢𑁆𑀭𑀚𑁄 𑀳𑀼𑀢𑁆𑀯𑀸,_  
_𑀫𑀺𑀕𑀺𑀬𑀸 𑀓𑀼𑀘𑁆𑀙𑀺𑀫𑁄𑀓𑁆𑀓𑀫𑀺𑀁𑁇_  


(164.)

7119\. _𑀢𑀤𑀸 𑀫𑀬𑀺 𑀓𑀼𑀘𑁆𑀙𑀺𑀕𑀢𑁂,_  
_𑀯𑀚𑁆𑀛𑀯𑀸𑀭𑁄 𑀉𑀧𑀝𑁆𑀞𑀺𑀢𑁄;_  
_𑀲𑀸𑀔𑁂𑀦 𑀘𑀢𑁆𑀢𑀸 𑀫𑁂 𑀫𑀸𑀢𑀸,_  
_𑀦𑀺𑀕𑁆𑀭𑁄𑀥𑀁 𑀲𑀭𑀡𑀁 𑀕𑀢𑀸𑁇_  


(165.)

7120\. _𑀢𑁂𑀦 𑀲𑀸 𑀫𑀺𑀕𑀭𑀸𑀚𑁂𑀦,_  
_𑀫𑀭𑀡𑀸 𑀧𑀭𑀺𑀫𑁄𑀘𑀺𑀢𑀸;_  
_𑀧𑀭𑀺𑀘𑁆𑀘𑀚𑀺𑀢𑁆𑀯𑀸 𑀲𑀧𑀸𑀡𑀁,_  
_𑀫𑀫𑁂𑀯𑀁 𑀑𑀯𑀤𑀻 𑀢𑀤𑀸𑁇_  


(166.)

7121\. _‘𑀦𑀺𑀕𑁆𑀭𑁄𑀥𑀫𑁂𑀯 𑀲𑁂𑀯𑁂𑀬𑁆𑀬,_  
_𑀦 𑀲𑀸𑀔𑀫𑀼𑀧𑀲𑀁𑀯𑀲𑁂;_  
_𑀦𑀺𑀕𑁆𑀭𑁄𑀥𑀲𑁆𑀫𑀺𑀁 𑀫𑀢𑀁 𑀲𑁂𑀬𑁆𑀬𑁄,_  
_𑀬𑀜𑁆𑀘𑁂 𑀲𑀸𑀔𑀫𑁆𑀳𑀺 𑀚𑀻𑀯𑀺𑀢𑀁’𑁇_  


(167.)

7122\. _𑀢𑁂𑀦𑀸𑀦𑀼𑀲𑀺𑀝𑁆𑀞𑀸 𑀫𑀺𑀕𑀬𑀽𑀣𑀧𑁂𑀦,_  
_𑀅𑀳𑀜𑁆𑀘 𑀫𑀸𑀢𑀸 𑀘 𑀢𑀣𑁂𑀢𑀭𑁂 𑀘;_  
_𑀆𑀕𑀫𑁆𑀫 𑀭𑀫𑁆𑀫𑀁 𑀢𑀼𑀲𑀺𑀢𑀸𑀥𑀺𑀯𑀸𑀲𑀁,_  
_𑀕𑀢𑀸 𑀧𑀯𑀸𑀲𑀁 𑀲𑀖𑀭𑀁 𑀬𑀣𑁂𑀯𑁇_  


(168.)

7123\. _𑀧𑀼𑀦𑁄 𑀓𑀲𑁆𑀲𑀧𑀯𑀻𑀭𑀲𑁆𑀲,_  
_𑀅𑀢𑁆𑀣𑀫𑁂𑀦𑁆𑀢𑀫𑁆𑀳𑀺 𑀲𑀸𑀲𑀦𑁂;_  
_𑀆𑀭𑀼𑀬𑁆𑀳 𑀲𑁂𑀮𑀲𑀺𑀔𑀭𑀁,_  
_𑀬𑀼𑀜𑁆𑀚𑀺𑀢𑁆𑀯𑀸 𑀚𑀺𑀦𑀲𑀸𑀲𑀦𑀁𑁇_  


(169.)

7124\. _𑀇𑀤𑀸𑀦𑀸𑀳𑀁 𑀭𑀸𑀚𑀕𑀳𑁂,_  
_𑀚𑀸𑀢𑁄 𑀲𑁂𑀝𑁆𑀞𑀺𑀓𑀼𑀮𑁂 𑀅𑀳𑀼𑀁;_  
_𑀆𑀧𑀦𑁆𑀦𑀲𑀢𑁆𑀢𑀸 𑀫𑁂 𑀫𑀸𑀢𑀸,_  
_𑀧𑀩𑁆𑀩𑀚𑀺 𑀅𑀦𑀕𑀸𑀭𑀺𑀬𑀁𑁇_  


(170.)

7125\. _𑀲𑀕𑀩𑁆𑀪𑀁 𑀢𑀁 𑀯𑀺𑀤𑀺𑀢𑁆𑀯𑀸𑀦,_  
_𑀤𑁂𑀯𑀤𑀢𑁆𑀢𑀫𑀼𑀧𑀸𑀦𑀬𑀼𑀁;_  
_𑀲𑁄 𑀅𑀯𑁄𑀘 ‘𑀯𑀺𑀦𑀸𑀲𑁂𑀣,_  
_𑀧𑀸𑀧𑀺𑀓𑀁 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀁 𑀇𑀫𑀁’𑁇_  


(171.)

7126\. _𑀇𑀤𑀸𑀦𑀺𑀧𑀺 𑀫𑀼𑀦𑀺𑀦𑁆𑀤𑁂𑀦,_  
_𑀚𑀺𑀦𑁂𑀦 𑀅𑀦𑀼𑀓𑀫𑁆𑀧𑀺𑀢𑀸;_  
_𑀲𑀼𑀔𑀺𑀦𑀻 𑀅𑀚𑀦𑀻 𑀫𑀬𑁆𑀳𑀁,_  
_𑀫𑀸𑀢𑀸 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀼𑀧𑀲𑁆𑀲𑀬𑁂𑁇_  


(172.)

7127\. _𑀢𑀁 𑀯𑀺𑀤𑀺𑀢𑁆𑀯𑀸 𑀫𑀳𑀻𑀧𑀸𑀮𑁄,_  
_𑀓𑁄𑀲𑀮𑁄 𑀫𑀁 𑀅𑀧𑁄𑀲𑀬𑀺;_  
_𑀓𑀼𑀫𑀸𑀭𑀧𑀭𑀺𑀳𑀸𑀭𑁂𑀦,_  
_𑀦𑀸𑀫𑁂𑀦𑀸𑀳𑀜𑁆𑀘 𑀓𑀲𑁆𑀲𑀧𑁄𑁇_  


(173.)

7128\. _𑀫𑀳𑀸𑀓𑀲𑁆𑀲𑀧𑀫𑀸𑀕𑀫𑁆𑀫,_  
_𑀅𑀳𑀁 𑀓𑀼𑀫𑀸𑀭𑀓𑀲𑁆𑀲𑀧𑁄;_  
_𑀯𑀫𑁆𑀫𑀺𑀓𑀲𑀤𑀺𑀲𑀁 𑀓𑀸𑀬𑀁,_  
_𑀲𑀼𑀢𑁆𑀯𑀸 𑀩𑀼𑀤𑁆𑀥𑁂𑀦 𑀤𑁂𑀲𑀺𑀢𑀁𑁇_  


(174.)

7129\. _𑀢𑀢𑁄 𑀘𑀺𑀢𑁆𑀢𑀁 𑀯𑀺𑀫𑀼𑀘𑁆𑀘𑀺 𑀫𑁂,_  
_𑀅𑀦𑀼𑀧𑀸𑀤𑀸𑀬 𑀲𑀩𑁆𑀩𑀲𑁄;_  
_𑀧𑀸𑀬𑀸𑀲𑀺𑀁 𑀤𑀫𑀬𑀺𑀢𑁆𑀯𑀸𑀳𑀁,_  
_𑀏𑀢𑀤𑀕𑁆𑀕𑀫𑀧𑀸𑀧𑀼𑀡𑀺𑀁𑁇_  


(175.)

7130\. _𑀓𑀺𑀮𑁂𑀲𑀸 𑀛𑀸𑀧𑀺𑀢𑀸 𑀫𑀬𑁆𑀳𑀁,_  
_…𑀧𑁂…_  
_𑀯𑀺𑀳𑀭𑀸𑀫𑀺 𑀅𑀦𑀸𑀲𑀯𑁄𑁇_  


(176.)

7131\. _𑀲𑁆𑀯𑀸𑀕𑀢𑀁 𑀯𑀢 𑀫𑁂 𑀆𑀲𑀺,_  
_…𑀧𑁂…_  
_𑀓𑀢𑀁 𑀩𑀼𑀤𑁆𑀥𑀲𑁆𑀲 𑀲𑀸𑀲𑀦𑀁𑁇_  


(177.)

7132\. _𑀧𑀝𑀺𑀲𑀫𑁆𑀪𑀺𑀤𑀸 𑀘𑀢𑀲𑁆𑀲𑁄,_  
_…𑀧𑁂…_  
_𑀓𑀢𑀁 𑀩𑀼𑀤𑁆𑀥𑀲𑁆𑀲 𑀲𑀸𑀲𑀦𑀁”𑁇_
