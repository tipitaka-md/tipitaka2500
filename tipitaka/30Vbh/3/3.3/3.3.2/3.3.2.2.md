# 3.3.2.2 Cūḷantaraduka

### Dhātuvibhaṅga

### Pañhāpucchaka

### Duka

### Cūḷantaraduka

694\. Sattarasa dhātuyo sappaccayā. Dhammadhātu siyā sappaccayā, siyā appaccayā. Sattarasa dhātuyo saṅkhatā. Dhammadhātu siyā saṅkhatā, siyā asaṅkhatā. Rūpadhātu sanidassanā. Sattarasa dhātuyo anidassanā. Dasa dhātuyo sappaṭighā. Aṭṭha dhātuyo appaṭighā. Dasa dhātuyo rūpā. Satta dhātuyo arūpā. Dhammadhātu siyā rūpā, siyā arūpā. Soḷasa dhātuyo lokiyā. Dve dhātuyo siyā lokiyā, siyā lokuttarā; kenaci viññeyyā, kenaci na viññeyyā.
