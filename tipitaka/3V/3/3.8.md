# 𑁩𑁇𑁮 𑀲𑀁𑀖𑀪𑁂𑀤𑁂𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀯𑀸𑀭

* 𑀯𑀲𑁆𑀲𑀽𑀧𑀦𑀸𑀬𑀺𑀓𑀓𑁆𑀔𑀦𑁆𑀥𑀓

* 𑀲𑀁𑀖𑀪𑁂𑀤𑁂𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀯𑀸𑀭

(202.)

896\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀧𑀲𑁆𑀲𑀢𑀺 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑁂𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀕𑀭𑀼𑀓𑁄 𑀔𑁄 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸 𑀫𑀬𑀺 𑀲𑀫𑁆𑀫𑀼𑀔𑀻𑀪𑀽𑀢𑁂 𑀲𑀁𑀖𑁄 𑀪𑀺𑀚𑁆𑀚𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

897\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀲𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸 𑀪𑀺𑀓𑁆𑀔𑀽 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑀻’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀕𑀭𑀼𑀓𑁄 𑀔𑁄 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸 𑀫𑀬𑀺 𑀲𑀫𑁆𑀫𑀼𑀔𑀻𑀪𑀽𑀢𑁂 𑀲𑀁𑀖𑁄 𑀪𑀺𑀚𑁆𑀚𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

898\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀲𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸 𑀪𑀺𑀓𑁆𑀔𑀽 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑀻’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑁂 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑁆𑀬𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀆𑀯𑀼𑀲𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸𑀬𑀲𑁆𑀫𑀦𑁆𑀢𑀸𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀫𑁂 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

899\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀲𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸 𑀪𑀺𑀓𑁆𑀔𑀽 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑀻’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑁂 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀦 𑀫𑀺𑀢𑁆𑀢𑀸; 𑀅𑀧𑀺 𑀘 𑀬𑁂 𑀢𑁂𑀲𑀁 𑀫𑀺𑀢𑁆𑀢𑀸, 𑀢𑁂 𑀫𑁂 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑁆𑀬𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺𑁇 𑀢𑁂 𑀯𑀼𑀢𑁆𑀢𑀸 𑀢𑁂 𑀯𑀓𑁆𑀔𑀦𑁆𑀢𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀆𑀯𑀼𑀲𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸𑀬𑀲𑁆𑀫𑀦𑁆𑀢𑀸𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀢𑁂𑀲𑀁 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

900\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀲𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑁂𑀳𑀺 𑀪𑀺𑀓𑁆𑀔𑀽𑀳𑀺 𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑁂 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑁆𑀬𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀆𑀯𑀼𑀲𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸𑀬𑀲𑁆𑀫𑀦𑁆𑀢𑀸𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀫𑁂 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

901\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀲𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑁂𑀳𑀺 𑀪𑀺𑀓𑁆𑀔𑀽𑀳𑀺 𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑁂 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀦 𑀫𑀺𑀢𑁆𑀢𑀸; 𑀅𑀧𑀺 𑀘 𑀬𑁂 𑀢𑁂𑀲𑀁 𑀫𑀺𑀢𑁆𑀢𑀸 𑀢𑁂 𑀫𑁂 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑁆𑀬𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺𑁇 𑀢𑁂 𑀯𑀼𑀢𑁆𑀢𑀸 𑀢𑁂 𑀯𑀓𑁆𑀔𑀦𑁆𑀢𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀆𑀯𑀼𑀲𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸𑀬𑀲𑁆𑀫𑀦𑁆𑀢𑀸𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀢𑁂𑀲𑀁 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

902\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀫𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑀻’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑀸 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀪𑀕𑀺𑀦𑀺𑀬𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸 𑀪𑀕𑀺𑀦𑀻𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀫𑁂 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

903\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀫𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀲𑀁𑀖𑀪𑁂𑀤𑀸𑀬 𑀧𑀭𑀓𑁆𑀓𑀫𑀦𑁆𑀢𑀻’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑀸 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀦 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀅𑀧𑀺 𑀘 𑀬𑀸 𑀢𑀸𑀲𑀁 𑀫𑀺𑀢𑁆𑀢𑀸, 𑀢𑀸 𑀫𑁂 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺𑁇 𑀢𑀸 𑀯𑀼𑀢𑁆𑀢𑀸 𑀢𑀸 𑀯𑀓𑁆𑀔𑀦𑁆𑀢𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀪𑀕𑀺𑀦𑀺𑀬𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸𑁇 𑀫𑀸 𑀪𑀕𑀺𑀦𑀻𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀢𑀸𑀲𑀁 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

904\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀫𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸𑀳𑀺 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀻𑀳𑀺 𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑀸 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀪𑀕𑀺𑀦𑀺𑀬𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸𑁇 𑀫𑀸 𑀪𑀕𑀺𑀦𑀻𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀫𑁂 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑁇

905\. 𑀇𑀥 𑀧𑀦, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀯𑀲𑁆𑀲𑀽𑀧𑀕𑀢𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀲𑀼𑀡𑀸𑀢𑀺—  ‘𑀅𑀫𑀼𑀓𑀲𑁆𑀫𑀺𑀁 𑀓𑀺𑀭 𑀆𑀯𑀸𑀲𑁂 𑀲𑀫𑁆𑀩𑀳𑀼𑀮𑀸𑀳𑀺 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀻𑀳𑀺 𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄’𑀢𑀺𑁇 𑀢𑀢𑁆𑀭 𑀘𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑁄 𑀏𑀯𑀁 𑀳𑁄𑀢𑀺—  ‘𑀢𑀸 𑀔𑁄 𑀫𑁂 𑀪𑀺𑀓𑁆𑀔𑀼𑀦𑀺𑀬𑁄 𑀦 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀅𑀧𑀺 𑀘 𑀬𑀸 𑀢𑀸𑀲𑀁 𑀫𑀺𑀢𑁆𑀢𑀸 𑀢𑀸 𑀫𑁂 𑀫𑀺𑀢𑁆𑀢𑀸𑁇 𑀢𑀸𑀳𑀁 𑀯𑀓𑁆𑀔𑀸𑀫𑀺𑁇 𑀢𑀸 𑀯𑀼𑀢𑁆𑀢𑀸 𑀢𑀸 𑀯𑀓𑁆𑀔𑀦𑁆𑀢𑀺 “𑀕𑀭𑀼𑀓𑁄 𑀔𑁄, 𑀪𑀕𑀺𑀦𑀺𑀬𑁄, 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀯𑀼𑀢𑁆𑀢𑁄 𑀪𑀕𑀯𑀢𑀸; 𑀫𑀸 𑀪𑀕𑀺𑀦𑀻𑀦𑀁 𑀲𑀁𑀖𑀪𑁂𑀤𑁄 𑀭𑀼𑀘𑁆𑀘𑀺𑀢𑁆𑀣𑀸”𑀢𑀺𑁇 𑀓𑀭𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺 𑀢𑀸𑀲𑀁 𑀯𑀘𑀦𑀁, 𑀲𑀼𑀲𑁆𑀲𑀽𑀲𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺, 𑀲𑁄𑀢𑀁 𑀑𑀤𑀳𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀻’𑀢𑀺, 𑀧𑀓𑁆𑀓𑀫𑀺𑀢𑀩𑁆𑀩𑀁𑁇 𑀅𑀦𑀸𑀧𑀢𑁆𑀢𑀺 𑀯𑀲𑁆𑀲𑀘𑁆𑀙𑁂𑀤𑀲𑁆𑀲𑀸”𑀢𑀺𑁇
