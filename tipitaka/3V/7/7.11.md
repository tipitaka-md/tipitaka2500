# 7.11 Karaṇīyadoḷasaka

### Kathinakkhandhaka

### Karaṇīyadoḷasaka

(320.)

1495. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. So taṃ cīvaraṃ kāreti. Tassa bhikkhuno niṭṭhānantiko kathinuddhāro. (1)

1496. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “nevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. Tassa bhikkhuno sanniṭṭhānantiko kathinuddhāro. (2)

1497. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. So taṃ cīvaraṃ kāreti. Tassa taṃ cīvaraṃ kayiramānaṃ nassati. Tassa bhikkhuno nāsanantiko kathinuddhāro. (3)

1498. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati. Tassa bahisīmagatassa cīvarāsā uppajjati. Tassa evaṃ hoti—  “idhevimaṃ cīvarāsaṃ payirupāsissaṃ, na paccessan”ti. So taṃ cīvarāsaṃ payirupāsati. Tassa sā cīvarāsā upacchijjati. Tassa bhikkhuno āsāvacchediko kathinuddhāro. (4)

1499. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressan”ti. So taṃ cīvaraṃ kāreti. Tassa bhikkhuno niṭṭhānantiko kathinuddhāro. (5)

1500. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “nevimaṃ cīvaraṃ kāressan”ti. Tassa bhikkhuno sanniṭṭhānantiko kathinuddhāro. (6)

1501. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressan”ti. So taṃ cīvaraṃ kāreti. Tassa taṃ cīvaraṃ kayiramānaṃ nassati. Tassa bhikkhuno nāsanantiko kathinuddhāro. (7)

1502. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. Tassa evaṃ hoti—  “idhevimaṃ cīvarāsaṃ payirupāsissan”ti. So taṃ cīvarāsaṃ payirupāsati. Tassa sā cīvarāsā upacchijjati. Tassa bhikkhuno āsāvacchediko kathinuddhāro. (8)

1503. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati anadhiṭṭhitena; nevassa hoti—  “paccessan”ti, na panassa hoti—  “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. So taṃ cīvaraṃ kāreti. Tassa bhikkhuno niṭṭhānantiko kathinuddhāro. (9)

1504. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati anadhiṭṭhitena; nevassa hoti—  “paccessan”ti, na panassa hoti—  “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “nevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. Tassa bhikkhuno sanniṭṭhānantiko kathinuddhāro. (10)

1505. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati anadhiṭṭhitena; nevassa hoti—  “paccessan”ti, na panassa hoti—  “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. So taṃ cīvarāsaṃ payirupāsati. Anāsāya labhati, āsāya na labhati. Tassa evaṃ hoti—  “idhevimaṃ cīvaraṃ kāressaṃ, na paccessan”ti. So taṃ cīvaraṃ kāreti. Tassa taṃ cīvaraṃ kayiramānaṃ nassati. Tassa bhikkhuno nāsanantiko kathinuddhāro. (11)

1506. Bhikkhu atthatakathino kenacideva karaṇīyena pakkamati anadhiṭṭhitena; nevassa hoti—  “paccessan”ti, na panassa hoti—  “na paccessan”ti. Tassa bahisīmagatassa cīvarāsā uppajjati. Tassa evaṃ hoti—  “idhevimaṃ cīvarāsaṃ payirupāsissaṃ, na paccessan”ti. So taṃ cīvarāsaṃ payirupāsati. Tassa sā cīvarāsā upacchijjati. Tassa bhikkhuno āsāvacchediko kathinuddhāro. (12)

===

1507. Karaṇīyadoḷasakaṃ niṭṭhitaṃ.
