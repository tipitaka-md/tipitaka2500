# 2.5 Uposathāgārakathā

* Uposathakkhandhaka

* Uposathāgārakathā

(141.)

596\. Tena kho pana samayena bhikkhū anupariveṇiyaṃ pātimokkhaṃ uddisanti asaṅketena. Āgantukā bhikkhū na jānanti—  “kattha vā ajjuposatho karīyissatī”ti. Bhagavato etamatthaṃ ārocesuṃ. “Na, bhikkhave, anupariveṇiyaṃ pātimokkhaṃ uddisitabbaṃ asaṅketena. Yo uddiseyya, āpatti dukkaṭassa. Anujānāmi, bhikkhave, uposathāgāraṃ sammannitvā uposathaṃ kātuṃ, yaṃ saṃgho ākaṅkhati vihāraṃ vā aḍḍhayogaṃ vā pāsādaṃ vā hammiyaṃ vā guhaṃ vā. Evañca pana, bhikkhave, sammannitabbaṃ. Byattena bhikkhunā paṭibalena saṃgho ñāpetabbo—

597\. ‘Suṇātu me, bhante, saṃgho. Yadi saṃghassa pattakallaṃ, saṃgho itthannāmaṃ vihāraṃ uposathāgāraṃ sammanneyya. Esā ñatti.

598\. Suṇātu me, bhante, saṃgho. Saṃgho itthannāmaṃ vihāraṃ uposathāgāraṃ sammannati. Yassāyasmato khamati itthannāmassa vihārassa uposathāgārassa sammuti, so tuṇhassa; yassa nakkhamati, so bhāseyya. Sammato saṃghena itthannāmo vihāro uposathāgāraṃ. Khamati saṃghassa, tasmā tuṇhī, evametaṃ dhārayāmī’”ti.

599\. Tena kho pana samayena aññatarasmiṃ āvāse dve uposathāgārāni sammatāni honti. Bhikkhū ubhayattha sannipatanti—  “idha uposatho karīyissati, idha uposatho karīyissatī”ti. Bhagavato etamatthaṃ ārocesuṃ. “Na, bhikkhave, ekasmiṃ āvāse dve uposathāgārāni sammannitabbāni. Yo sammanneyya, āpatti dukkaṭassa. Anujānāmi, bhikkhave, ekaṃ samūhanitvā ekattha uposathaṃ kātuṃ. Evañca pana, bhikkhave, samūhantabbaṃ. Byattena bhikkhunā paṭibalena saṃgho ñāpetabbo—

600\. ‘Suṇātu me, bhante, saṃgho. Yadi saṃghassa pattakallaṃ, saṃgho itthannāmaṃ uposathāgāraṃ samūhaneyya. Esā ñatti.

601\. Suṇātu me, bhante, saṃgho. Saṃgho itthannāmaṃ uposathāgāraṃ samūhanati. Yassāyasmato khamati itthannāmassa uposathāgārassa samugghāto, so tuṇhassa; yassa nakkhamati, so bhāseyya. Samūhataṃ saṃghena itthannāmaṃ uposathāgāraṃ. Khamati saṃghassa, tasmā tuṇhī, evametaṃ dhārayāmī’”ti.
