### Nahetuduka

(65.)

1142. Nahetupaccayā ārammaṇe dve, anantare dve, samanantare dve…pe…  kamme dve, vipāke ekaṃ, āhāre dve…pe…  magge ekaṃ, sampayutte dve, vippayutte dve…pe…  avigate dve.
