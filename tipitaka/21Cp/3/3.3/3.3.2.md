# 3.3.2 Saccatāpasacariya

### Yudhañjayavagga

### Saccapāramī

### Saccatāpasacariya

(71.)

346. _“Punāparaṃ yadā homi,_  
_Tāpaso saccasavhayo;_  
_Saccena lokaṃ pālesiṃ,_  
_Samaggaṃ janamakāsahan”ti._  


===

347. Saccatāpasacariyaṃ aṭṭhamaṃ.
