# 𑁩𑁇𑁨𑁇𑁩 𑀕𑀺𑀜𑁆𑀚𑀓𑀸𑀯𑀲𑀣𑀲𑀼𑀢𑁆𑀢

* 𑀥𑀸𑀢𑀼𑀲𑀁𑀬𑀼𑀢𑁆𑀢

* 𑀤𑀼𑀢𑀺𑀬𑀯𑀕𑁆𑀕

* 𑀕𑀺𑀜𑁆𑀚𑀓𑀸𑀯𑀲𑀣𑀲𑀼𑀢𑁆𑀢

(97.)

700\. 𑀏𑀓𑀁 𑀲𑀫𑀬𑀁 𑀪𑀕𑀯𑀸 𑀜𑀸𑀢𑀺𑀓𑁂 𑀯𑀺𑀳𑀭𑀢𑀺 𑀕𑀺𑀜𑁆𑀚𑀓𑀸𑀯𑀲𑀣𑁂𑁇 𑀢𑀢𑁆𑀭 𑀔𑁄 𑀪𑀕𑀯𑀸 𑀪𑀺𑀓𑁆𑀔𑀽 𑀆𑀫𑀦𑁆𑀢𑁂𑀲𑀺—  “𑀪𑀺𑀓𑁆𑀔𑀯𑁄”𑀢𑀺𑁇 “𑀪𑀤𑀦𑁆𑀢𑁂”𑀢𑀺 𑀢𑁂 𑀪𑀺𑀓𑁆𑀔𑀽 𑀪𑀕𑀯𑀢𑁄 𑀧𑀘𑁆𑀘𑀲𑁆𑀲𑁄𑀲𑀼𑀁𑁇 𑀪𑀕𑀯𑀸 𑀏𑀢𑀤𑀯𑁄𑀘—

701\. “𑀥𑀸𑀢𑀼𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀧𑀝𑀺𑀘𑁆𑀘 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀲𑀜𑁆𑀜𑀸, 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀤𑀺𑀝𑁆𑀞𑀺, 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀯𑀺𑀢𑀓𑁆𑀓𑁄”𑀢𑀺𑁇 𑀏𑀯𑀁 𑀯𑀼𑀢𑁆𑀢𑁂, 𑀆𑀬𑀲𑁆𑀫𑀸 𑀓𑀘𑁆𑀘𑀸𑀦𑁄 𑀪𑀕𑀯𑀦𑁆𑀢𑀁 𑀏𑀢𑀤𑀯𑁄𑀘—  “𑀬𑀸𑀬𑀁, 𑀪𑀦𑁆𑀢𑁂, 𑀤𑀺𑀝𑁆𑀞𑀺—  ‘𑀅𑀲𑀫𑁆𑀫𑀸𑀲𑀫𑁆𑀩𑀼𑀤𑁆𑀥𑁂𑀲𑀼 𑀲𑀫𑁆𑀫𑀸𑀲𑀫𑁆𑀩𑀼𑀤𑁆𑀥𑀸’𑀢𑀺, 𑀅𑀬𑀁 𑀦𑀼 𑀔𑁄, 𑀪𑀦𑁆𑀢𑁂, 𑀤𑀺𑀝𑁆𑀞𑀺 𑀓𑀺𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀻”𑀢𑀺?

702\. “𑀫𑀳𑀢𑀺 𑀔𑁄 𑀏𑀲𑀸, 𑀓𑀘𑁆𑀘𑀸𑀦, 𑀥𑀸𑀢𑀼 𑀬𑀤𑀺𑀤𑀁 𑀅𑀯𑀺𑀚𑁆𑀚𑀸𑀥𑀸𑀢𑀼𑁇 𑀳𑀻𑀦𑀁, 𑀓𑀘𑁆𑀘𑀸𑀦, 𑀥𑀸𑀢𑀼𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀳𑀻𑀦𑀸 𑀲𑀜𑁆𑀜𑀸, 𑀳𑀻𑀦𑀸 𑀤𑀺𑀝𑁆𑀞𑀺, 𑀳𑀻𑀦𑁄 𑀯𑀺𑀢𑀓𑁆𑀓𑁄, 𑀳𑀻𑀦𑀸 𑀘𑁂𑀢𑀦𑀸, 𑀳𑀻𑀦𑀸 𑀧𑀢𑁆𑀣𑀦𑀸, 𑀳𑀻𑀦𑁄 𑀧𑀡𑀺𑀥𑀺, 𑀳𑀻𑀦𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄, 𑀳𑀻𑀦𑀸 𑀯𑀸𑀘𑀸; 𑀳𑀻𑀦𑀁 𑀆𑀘𑀺𑀓𑁆𑀔𑀢𑀺 𑀤𑁂𑀲𑁂𑀢𑀺 𑀧𑀜𑁆𑀜𑀧𑁂𑀢𑀺 𑀧𑀝𑁆𑀞𑀧𑁂𑀢𑀺 𑀯𑀺𑀯𑀭𑀢𑀺 𑀯𑀺𑀪𑀚𑀢𑀺 𑀉𑀢𑁆𑀢𑀸𑀦𑀻𑀓𑀭𑁄𑀢𑀺; 𑀳𑀻𑀦𑀸 𑀢𑀲𑁆𑀲 𑀉𑀧𑀧𑀢𑁆𑀢𑀻𑀢𑀺 𑀯𑀤𑀸𑀫𑀺𑁇

703\. 𑀫𑀚𑁆𑀛𑀺𑀫𑀁, 𑀓𑀘𑁆𑀘𑀸𑀦, 𑀥𑀸𑀢𑀼𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀲𑀜𑁆𑀜𑀸, 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀤𑀺𑀝𑁆𑀞𑀺, 𑀫𑀚𑁆𑀛𑀺𑀫𑁄 𑀯𑀺𑀢𑀓𑁆𑀓𑁄, 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀘𑁂𑀢𑀦𑀸, 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀧𑀢𑁆𑀣𑀦𑀸, 𑀫𑀚𑁆𑀛𑀺𑀫𑁄 𑀧𑀡𑀺𑀥𑀺, 𑀫𑀚𑁆𑀛𑀺𑀫𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄, 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀯𑀸𑀘𑀸; 𑀫𑀚𑁆𑀛𑀺𑀫𑀁 𑀆𑀘𑀺𑀓𑁆𑀔𑀢𑀺 𑀤𑁂𑀲𑁂𑀢𑀺 𑀧𑀜𑁆𑀜𑀧𑁂𑀢𑀺 𑀧𑀝𑁆𑀞𑀧𑁂𑀢𑀺 𑀯𑀺𑀯𑀭𑀢𑀺 𑀯𑀺𑀪𑀚𑀢𑀺 𑀉𑀢𑁆𑀢𑀸𑀦𑀻𑀓𑀭𑁄𑀢𑀺; 𑀫𑀚𑁆𑀛𑀺𑀫𑀸 𑀢𑀲𑁆𑀲 𑀉𑀧𑀧𑀢𑁆𑀢𑀻𑀢𑀺 𑀯𑀤𑀸𑀫𑀺𑁇

704\. 𑀧𑀡𑀻𑀢𑀁, 𑀓𑀘𑁆𑀘𑀸𑀦, 𑀥𑀸𑀢𑀼𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀉𑀧𑁆𑀧𑀚𑁆𑀚𑀢𑀺 𑀧𑀡𑀻𑀢𑀸 𑀲𑀜𑁆𑀜𑀸, 𑀧𑀡𑀻𑀢𑀸 𑀤𑀺𑀝𑁆𑀞𑀺, 𑀧𑀡𑀻𑀢𑁄 𑀯𑀺𑀢𑀓𑁆𑀓𑁄, 𑀧𑀡𑀻𑀢𑀸 𑀘𑁂𑀢𑀦𑀸, 𑀧𑀡𑀻𑀢𑀸 𑀧𑀢𑁆𑀣𑀦𑀸, 𑀧𑀡𑀻𑀢𑁄 𑀧𑀡𑀺𑀥𑀺, 𑀧𑀡𑀻𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄, 𑀧𑀡𑀻𑀢𑀸 𑀯𑀸𑀘𑀸; 𑀧𑀡𑀻𑀢𑀁 𑀆𑀘𑀺𑀓𑁆𑀔𑀢𑀺 𑀤𑁂𑀲𑁂𑀢𑀺 𑀧𑀜𑁆𑀜𑀧𑁂𑀢𑀺 𑀧𑀝𑁆𑀞𑀧𑁂𑀢𑀺 𑀯𑀺𑀯𑀭𑀢𑀺 𑀯𑀺𑀪𑀚𑀢𑀺 𑀉𑀢𑁆𑀢𑀸𑀦𑀻𑀓𑀭𑁄𑀢𑀺; 𑀧𑀡𑀻𑀢𑀸 𑀢𑀲𑁆𑀲 𑀉𑀧𑀧𑀢𑁆𑀢𑀻𑀢𑀺 𑀯𑀤𑀸𑀫𑀻”𑀢𑀺𑁇
