# 𑁩𑁇𑁨𑁇𑁧 𑀲𑀢𑁆𑀢𑀥𑀸𑀢𑀼𑀲𑀼𑀢𑁆𑀢

### 𑀥𑀸𑀢𑀼𑀲𑀁𑀬𑀼𑀢𑁆𑀢

### 𑀤𑀼𑀢𑀺𑀬𑀯𑀕𑁆𑀕

### 𑀲𑀢𑁆𑀢𑀥𑀸𑀢𑀼𑀲𑀼𑀢𑁆𑀢

(95.)

683. 𑀲𑀸𑀯𑀢𑁆𑀣𑀺𑀬𑀁 𑀯𑀺𑀳𑀭𑀢𑀺𑁇 “𑀲𑀢𑁆𑀢𑀺𑀫𑀸, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀥𑀸𑀢𑀼𑀬𑁄𑁇 𑀓𑀢𑀫𑀸 𑀲𑀢𑁆𑀢? 𑀆𑀪𑀸𑀥𑀸𑀢𑀼, 𑀲𑀼𑀪𑀥𑀸𑀢𑀼, 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼, 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼, 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼, 𑀦𑁂𑀯𑀲𑀜𑁆𑀜𑀸𑀦𑀸𑀲𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼, 𑀲𑀜𑁆𑀜𑀸𑀯𑁂𑀤𑀬𑀺𑀢𑀦𑀺𑀭𑁄𑀥𑀥𑀸𑀢𑀼—  𑀇𑀫𑀸 𑀔𑁄, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀲𑀢𑁆𑀢 𑀥𑀸𑀢𑀼𑀬𑁄”𑀢𑀺𑁇

684. 𑀏𑀯𑀁 𑀯𑀼𑀢𑁆𑀢𑁂, 𑀅𑀜𑁆𑀜𑀢𑀭𑁄 𑀪𑀺𑀓𑁆𑀔𑀼 𑀪𑀕𑀯𑀦𑁆𑀢𑀁 𑀏𑀢𑀤𑀯𑁄𑀘—  “𑀬𑀸 𑀘𑀸𑀬𑀁, 𑀪𑀦𑁆𑀢𑁂, 𑀆𑀪𑀸𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀲𑀼𑀪𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀦𑁂𑀯𑀲𑀜𑁆𑀜𑀸𑀦𑀸𑀲𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀲𑀜𑁆𑀜𑀸𑀯𑁂𑀤𑀬𑀺𑀢𑀦𑀺𑀭𑁄𑀥𑀥𑀸𑀢𑀼—  𑀇𑀫𑀸 𑀦𑀼 𑀔𑁄, 𑀪𑀦𑁆𑀢𑁂, 𑀥𑀸𑀢𑀼𑀬𑁄 𑀓𑀺𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀦𑁆𑀢𑀻”𑀢𑀺?

685. “𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀆𑀪𑀸𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀅𑀦𑁆𑀥𑀓𑀸𑀭𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀲𑀼𑀪𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀅𑀲𑀼𑀪𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀭𑀽𑀧𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀦𑁂𑀯𑀲𑀜𑁆𑀜𑀸𑀦𑀸𑀲𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀺𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀲𑀜𑁆𑀜𑀸𑀯𑁂𑀤𑀬𑀺𑀢𑀦𑀺𑀭𑁄𑀥𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀦𑀺𑀭𑁄𑀥𑀁 𑀧𑀝𑀺𑀘𑁆𑀘 𑀧𑀜𑁆𑀜𑀸𑀬𑀢𑀻”𑀢𑀺𑁇

686. “𑀬𑀸 𑀘𑀸𑀬𑀁, 𑀪𑀦𑁆𑀢𑁂, 𑀆𑀪𑀸𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀲𑀼𑀪𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀦𑁂𑀯𑀲𑀜𑁆𑀜𑀸𑀦𑀸𑀲𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀲𑀜𑁆𑀜𑀸𑀯𑁂𑀤𑀬𑀺𑀢𑀦𑀺𑀭𑁄𑀥𑀥𑀸𑀢𑀼—  𑀇𑀫𑀸 𑀦𑀼 𑀔𑁄, 𑀪𑀦𑁆𑀢𑁂, 𑀥𑀸𑀢𑀼𑀬𑁄 𑀓𑀣𑀁 𑀲𑀫𑀸𑀧𑀢𑁆𑀢𑀺 𑀧𑀢𑁆𑀢𑀩𑁆𑀩𑀸”𑀢𑀺?

687. “𑀬𑀸 𑀘𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀆𑀪𑀸𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀲𑀼𑀪𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀸𑀲𑀸𑀦𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀯𑀺𑀜𑁆𑀜𑀸𑀡𑀜𑁆𑀘𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼 𑀬𑀸 𑀘 𑀆𑀓𑀺𑀜𑁆𑀘𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀇𑀫𑀸 𑀥𑀸𑀢𑀼𑀬𑁄 𑀲𑀜𑁆𑀜𑀸𑀲𑀫𑀸𑀧𑀢𑁆𑀢𑀺 𑀧𑀢𑁆𑀢𑀩𑁆𑀩𑀸𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀦𑁂𑀯𑀲𑀜𑁆𑀜𑀸𑀦𑀸𑀲𑀜𑁆𑀜𑀸𑀬𑀢𑀦𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀲𑀗𑁆𑀔𑀸𑀭𑀸𑀯𑀲𑁂𑀲𑀲𑀫𑀸𑀧𑀢𑁆𑀢𑀺 𑀧𑀢𑁆𑀢𑀩𑁆𑀩𑀸𑁇 𑀬𑀸𑀬𑀁, 𑀪𑀺𑀓𑁆𑀔𑀼, 𑀲𑀜𑁆𑀜𑀸𑀯𑁂𑀤𑀬𑀺𑀢𑀦𑀺𑀭𑁄𑀥𑀥𑀸𑀢𑀼—  𑀅𑀬𑀁 𑀥𑀸𑀢𑀼 𑀦𑀺𑀭𑁄𑀥𑀲𑀫𑀸𑀧𑀢𑁆𑀢𑀺 𑀧𑀢𑁆𑀢𑀩𑁆𑀩𑀸”𑀢𑀺𑁇
