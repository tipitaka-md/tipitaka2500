# 3.3 Anāgatavāra

### Pariññāvāra

### Anāgatavāra

(167.)

801. »  Yo dukkhasaccaṃ parijānissati so samudayasaccaṃ pajahissatīti? Āmantā.

802. «  Yo vā pana…pe… ? Āmantā.

803. »  Yo dukkhasaccaṃ na parijānissati so samudayasaccaṃ nappajahissatīti? Āmantā.

804. «  Yo vā pana…pe… ? Āmantā.
