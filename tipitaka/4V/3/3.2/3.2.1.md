# 3.2.1 Agghasamodhānaparivāsa

* Samuccayakkhandhaka

* Parivāsa

* Agghasamodhānaparivāsa

(134.)

615\. Tena kho pana samayena aññataro bhikkhu sambahulā saṃghādisesā āpattiyo āpanno hoti—  ekā āpatti ekāhappaṭicchannā, ekā āpatti dvīhappaṭicchannā, ekā āpatti tīhappaṭicchannā, ekā āpatti catūhappaṭicchannā, ekā āpatti pañcāhappaṭicchannā, ekā āpatti chāhappaṭicchannā, ekā āpatti sattāhappaṭicchannā, ekā āpatti aṭṭhāhappaṭicchannā, ekā āpatti navāhappaṭicchannā, ekā āpatti dasāhappaṭicchannā. So bhikkhūnaṃ ārocesi—  “ahaṃ, āvuso, sambahulā saṃghādisesā āpattiyo āpajjiṃ—  ekā āpatti ekāhappaṭicchannā…pe…  ekā āpatti dasāhappaṭicchannā. Kathaṃ nu kho mayā paṭipajjitabban”ti? Bhagavato etamatthaṃ ārocesuṃ. “Tena hi, bhikkhave, saṃgho tassa bhikkhuno tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ detu. Evañca pana, bhikkhave, dātabbo—

616\. Tena, bhikkhave, bhikkhunā saṃghaṃ upasaṅkamitvā…pe…  evamassa vacanīyo—  ‘ahaṃ, bhante, sambahulā saṃghādisesā āpattiyo āpajjiṃ—  ekā āpatti ekāhappaṭicchannā…pe…  ekā āpatti dasāhappaṭicchannā. Sohaṃ, bhante, saṃghaṃ tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ yācāmī’ti. Dutiyampi yācitabbo. Tatiyampi yācitabbo. Byattena bhikkhunā paṭibalena saṃgho ñāpetabbo—

(135.)

617\. ‘Suṇātu me, bhante, saṃgho. Ayaṃ itthannāmo bhikkhu sambahulā saṃghādisesā āpattiyo āpajji—  ekā āpatti ekāhappaṭicchannā…pe…  ekā āpatti dasāhappaṭicchannā. So saṃghaṃ tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ yācati. Yadi saṃghassa pattakallaṃ, saṃgho itthannāmassa bhikkhuno tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ dadeyya. Esā ñatti.

618\. Suṇātu me, bhante, saṃgho. Ayaṃ itthannāmo bhikkhu sambahulā saṃghādisesā āpattiyo āpajji—  ekā āpatti ekāhappaṭicchannā…pe…  ekā āpatti dasāhappaṭicchannā. So saṃghaṃ tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ yācati. Saṃgho itthannāmassa bhikkhuno tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāsaṃ deti. Yassāyasmato khamati itthannāmassa bhikkhuno tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā; tassā agghena samodhānaparivāsassa dānaṃ, so tuṇhassa; yassa nakkhamati, so bhāseyya.

619\. Dutiyampi etamatthaṃ vadāmi…pe…  tatiyampi etamatthaṃ vadāmi…pe… .

620\. Dinno saṃghena itthannāmassa bhikkhuno tāsaṃ āpattīnaṃ yā āpatti dasāhappaṭicchannā tassā agghena samodhānaparivāso. Khamati saṃghassa, tasmā tuṇhī, evametaṃ dhārayāmī’”ti.
