# 3.2.6 Kolaṃkolakathā

### Anusayapaṇṇāsaka

### Dvādasamavagga

### Kolaṃkolakathā

(646.)

2119. × Na vattabbaṃ—  “kolaṃkolo puggalo kolaṃkolatāniyato”ti? Āmantā. Nanu so kolaṃkoloti? Āmantā. Hañci so kolaṃkolo, tena vata re vattabbe—  “kolaṃkolo puggalo kolaṃkolatāniyato”ti.

===

2120. Kolaṃkolakathā niṭṭhitā.
