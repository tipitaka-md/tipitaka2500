# 3.2.7 Ekabījīkathā

### Anusayapaṇṇāsaka

### Dvādasamavagga

### Ekabījīkathā

(647.)

2121. × Na vattabbaṃ—  “ekabījī puggalo ekabījitāniyato”ti? Āmantā. Nanu so ekabījīti? Āmantā. Hañci so ekabījī, tena vata re vattabbe—  “ekabījī puggalo ekabījitāniyato”ti.

===

2122. Ekabījīkathā niṭṭhitā.
