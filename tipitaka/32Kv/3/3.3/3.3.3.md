# 𑁩𑁇𑁩𑁇𑁩 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑀓𑀣𑀸

* 𑀅𑀦𑀼𑀲𑀬𑀧𑀡𑁆𑀡𑀸𑀲𑀓

* 𑀢𑁂𑀭𑀲𑀫𑀯𑀕𑁆𑀕

* 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑀓𑀣𑀸

(660.)

2150\. 𑁆 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀫𑀺𑀘𑁆𑀙𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀜𑁆𑀘 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀜𑁆𑀘 𑀉𑀪𑁄 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸𑀢𑀺? 𑀦 𑀳𑁂𑀯𑀁 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂…𑀧𑁂…

2151\. 𑁆 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀦𑀦𑀼 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀬𑀼𑀢𑁆𑀢𑀁 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀉𑀧𑁆𑀧𑀸𑀤𑀺𑀢𑀁 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀚𑀦𑀺𑀢𑀦𑁆𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀳𑀜𑁆𑀘𑀺 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀬𑀼𑀢𑁆𑀢𑀁 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀉𑀧𑁆𑀧𑀸𑀤𑀺𑀢𑀁 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀚𑀦𑀺𑀢𑀁, 𑀦𑁄 𑀘 𑀯𑀢 𑀭𑁂 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂—  “𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸”𑀢𑀺𑁇

(661.)

2152\. ๐ 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀅𑀪𑀩𑁆𑀩𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑀺𑀢𑀼𑀦𑁆𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀫𑀸𑀢𑀸 𑀚𑀻𑀯𑀺𑀢𑀸 𑀯𑁄𑀭𑁄𑀧𑀺𑀢𑀸…  𑀧𑀺𑀢𑀸 𑀚𑀻𑀯𑀺𑀢𑀸 𑀯𑁄𑀭𑁄𑀧𑀺𑀢𑁄…  𑀅𑀭𑀳𑀸 𑀚𑀻𑀯𑀺𑀢𑀸 𑀯𑁄𑀭𑁄𑀧𑀺𑀢𑁄…  𑀤𑀼𑀝𑁆𑀞𑁂𑀦 𑀘𑀺𑀢𑁆𑀢𑁂𑀦 𑀢𑀣𑀸𑀕𑀢𑀲𑁆𑀲 𑀮𑁄𑀳𑀺𑀢𑀁 𑀉𑀧𑁆𑀧𑀸𑀤𑀺𑀢𑀁…  𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄𑀢𑀺? 𑀦 𑀳𑁂𑀯𑀁 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂…𑀧𑁂… 𑁇

2153\. ๐ 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀝𑀺𑀲𑀁𑀳𑀭𑀺𑀢𑁆𑀯𑀸 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁄𑀤𑁂𑀢𑁆𑀯𑀸 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁂𑀢𑁆𑀯𑀸 𑀅𑀪𑀩𑁆𑀩𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑀺𑀢𑀼𑀦𑁆𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀫𑀸𑀢𑀸 𑀚𑀻𑀯𑀺𑀢𑀸 𑀯𑁄𑀭𑁄𑀧𑀺𑀢𑀸…  𑀧𑀺𑀢𑀸 𑀚𑀻𑀯𑀺𑀢𑀸 𑀯𑁄𑀭𑁄𑀧𑀺𑀢𑁄…𑀧𑁂…  𑀲𑀁𑀖𑁄 𑀪𑀺𑀦𑁆𑀦𑁄𑀢𑀺? 𑀦 𑀳𑁂𑀯𑀁 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂…𑀧𑁂… 𑁇

2154\. ๐ 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀝𑀺𑀲𑀁𑀳𑀭𑀺𑀢𑁆𑀯𑀸 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁄𑀤𑁂𑀢𑁆𑀯𑀸 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁂𑀢𑁆𑀯𑀸 𑀅𑀪𑀩𑁆𑀩𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑀺𑀢𑀼𑀦𑁆𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀦𑀦𑀼 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀝𑀺𑀲𑀁𑀳𑀝𑀁 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁄𑀤𑀺𑀢𑀁 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑀻𑀢𑀦𑁆𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀳𑀜𑁆𑀘𑀺 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀝𑀺𑀲𑀁𑀳𑀝𑀁 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁄𑀤𑀺𑀢𑀁 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑀻𑀢𑀁, 𑀦𑁄 𑀘 𑀯𑀢 𑀭𑁂 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂—  “𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀝𑀺𑀲𑀁𑀳𑀭𑀺𑀢𑁆𑀯𑀸 𑀓𑀼𑀓𑁆𑀓𑀼𑀘𑁆𑀘𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁄𑀤𑁂𑀢𑁆𑀯𑀸 𑀯𑀺𑀧𑁆𑀧𑀝𑀺𑀲𑀸𑀭𑀺𑀬𑀁 𑀧𑀝𑀺𑀯𑀺𑀦𑁂𑀢𑁆𑀯𑀸 𑀅𑀪𑀩𑁆𑀩𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑀺𑀢𑀼𑀦𑁆”𑀢𑀺𑁇

(662.)

2155\. 𑁆 𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀦𑀦𑀼 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀆𑀲𑀻𑀢𑀺? 𑀆𑀫𑀦𑁆𑀢𑀸𑁇 𑀳𑀜𑁆𑀘𑀺 𑀢𑀁 𑀓𑀫𑁆𑀫𑀁 𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀆𑀲𑀺, 𑀦𑁄 𑀘 𑀯𑀢 𑀭𑁂 𑀯𑀢𑁆𑀢𑀩𑁆𑀩𑁂—  “𑀅𑀦𑀦𑁆𑀢𑀭𑀸𑀧𑀬𑀼𑀢𑁆𑀢𑁄 𑀧𑀼𑀕𑁆𑀕𑀮𑁄 𑀲𑀫𑁆𑀫𑀢𑁆𑀢𑀦𑀺𑀬𑀸𑀫𑀁 𑀑𑀓𑁆𑀓𑀫𑁂𑀬𑁆𑀬𑀸”𑀢𑀺𑁇
