# Hetu

### Cūḷantaraduka, Kusalattika

### Sanidassanaduka, Kusalattika

### Abyākatapada

### Hetu

(19.)

1503. Anidassanaṃ abyākataṃ dhammaṃ paṭicca anidassano abyākato dhammo uppajjati hetupaccayā. (Saṃkhittaṃ.)

(20.)

1504. Hetuyā tīṇi, ārammaṇe ekaṃ, adhipatiyā tīṇi…pe…  avigate tīṇi. (Saṃkhittaṃ.)

1505. Nahetuyā tīṇi (sabbattha tīṇi), nakamme tīṇi, navipāke tīṇi, naāhāre tīṇi…pe…  novigate tīṇi. (Saṃkhittaṃ.)

1506. (Sahajātavāropi paccayavāropi nissayavāropi saṃsaṭṭhavāropi sampayuttavāropi paṭiccavārasadisā vitthāretabbā.)
