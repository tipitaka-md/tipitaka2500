# Paccanīyuddhāra

* Hetugocchaka, Kusalattika

* Hetusampayuttaduka, Kusalattika

* Abyākatapada

* Pañhāvāra

* Paccanīyuddhāra

(45.)

1356\. Hetusampayutto abyākato dhammo hetusampayuttassa abyākatassa dhammassa ārammaṇapaccayena paccayo…  sahajātapaccayena paccayo…  upanissayapaccayena paccayo.

(46.)

1357\. Nahetuyā satta, naārammaṇe satta. (Saṃkhittaṃ.)

1358\. Hetupaccayā naārammaṇe tīṇi. (Saṃkhittaṃ.)

1359\. Nahetupaccayā ārammaṇe cattāri. (Saṃkhittaṃ.)

1360\. (Yathā kusalattike pañhāvārassa anulomampi paccanīyampi anulomapaccanīyampi paccanīyānulomampi gaṇitaṃ, evaṃ gaṇetabbaṃ.)

---

1361\. Hetusampayuttadukakusalattikaṃ niṭṭhitaṃ.
