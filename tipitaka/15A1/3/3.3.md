# 3.3

* Akammaniyavagga

(23.)

49\. “Nāhaṃ, bhikkhave, aññaṃ ekadhammampi samanupassāmi yaṃ evaṃ abhāvitaṃ mahato anatthāya saṃvattati yathayidaṃ, bhikkhave, cittaṃ. Cittaṃ, bhikkhave, abhāvitaṃ mahato anatthāya saṃvattatī”ti.

---

50\. Tatiyaṃ.
