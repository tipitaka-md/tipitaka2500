# 3.1

### Akammaniyavagga

(21.)

45. “Nāhaṃ, bhikkhave, aññaṃ ekadhammampi samanupassāmi yaṃ evaṃ abhāvitaṃ akammaniyaṃ hoti yathayidaṃ, bhikkhave, cittaṃ. Cittaṃ, bhikkhave, abhāvitaṃ akammaniyaṃ hotī”ti.

===

46. Paṭhamaṃ.
