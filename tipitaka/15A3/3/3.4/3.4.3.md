# 3.4.3 Mittasutta

### Tatiyapaṇṇāsaka

### Yodhājīvavagga

### Mittasutta

(136.)

1022. “Tīhi, bhikkhave, aṅgehi samannāgato mitto sevitabbo. Katamehi tīhi? ( ) Duddadaṃ dadāti, dukkaraṃ karoti, dukkhamaṃ khamati—  imehi kho, bhikkhave, tīhi aṅgehi samannāgato mitto sevitabbo”ti.

===

1023. Tatiyaṃ.
