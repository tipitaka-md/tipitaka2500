# 𑁩𑁇𑁩𑁇𑁮 𑀫𑀳𑀺𑀁𑀲𑀭𑀸𑀚𑀚𑀸𑀢𑀓

### 𑀢𑀺𑀓𑀦𑀺𑀧𑀸𑀢

### 𑀉𑀤𑀧𑀸𑀦𑀯𑀕𑁆𑀕

### 𑀫𑀳𑀺𑀁𑀲𑀭𑀸𑀚𑀚𑀸𑀢𑀓

(82.)

782\. _“𑀓𑀺𑀫𑀢𑁆𑀣𑀫𑀪𑀺𑀲𑀦𑁆𑀥𑀸𑀬,_  
_𑀮𑀳𑀼𑀘𑀺𑀢𑁆𑀢𑀲𑁆𑀲 𑀤𑀼𑀩𑁆𑀪𑀺𑀦𑁄;_  
_𑀲𑀩𑁆𑀩𑀓𑀸𑀫𑀤𑀤𑀲𑁆𑀲𑁂𑀯,_  
_𑀇𑀫𑀁 𑀤𑀼𑀓𑁆𑀔𑀁 𑀢𑀺𑀢𑀺𑀓𑁆𑀔𑀲𑀺𑁇_  


(83.)

783\. _𑀲𑀺𑀗𑁆𑀕𑁂𑀦 𑀦𑀺𑀳𑀦𑀸𑀳𑁂𑀢𑀁,_  
_𑀧𑀤𑀲𑀸 𑀘 𑀅𑀥𑀺𑀝𑁆𑀞𑀳;_  
_𑀪𑀺𑀬𑁆𑀬𑁄 𑀩𑀸𑀮𑀸 𑀧𑀓𑀼𑀚𑁆𑀛𑁂𑀬𑁆𑀬𑀼𑀁,_  
_𑀦𑁄 𑀘𑀲𑁆𑀲 𑀧𑀝𑀺𑀲𑁂𑀥𑀓𑁄”𑁇_  


(84.)

784\. _“𑀫𑀫𑁂𑀯𑀸𑀬𑀁 𑀫𑀜𑁆𑀜𑀫𑀸𑀦𑁄,_  
_𑀅𑀜𑁆𑀜𑁂𑀧𑁂𑀯𑀁 𑀓𑀭𑀺𑀲𑁆𑀲𑀢𑀺;_  
_𑀢𑁂 𑀦𑀁 𑀢𑀢𑁆𑀣 𑀯𑀥𑀺𑀲𑁆𑀲𑀦𑁆𑀢𑀺,_  
_𑀲𑀸 𑀫𑁂 𑀫𑀼𑀢𑁆𑀢𑀺 𑀪𑀯𑀺𑀲𑁆𑀲𑀢𑀻”𑀢𑀺𑁇_  


---
