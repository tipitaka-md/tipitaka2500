# 𑁧𑁇𑁩𑁇𑁩 𑀪𑁄𑀚𑀸𑀚𑀸𑀦𑀻𑀬𑀚𑀸𑀢𑀓

### 𑀪𑁄𑀚𑀸𑀚𑀸𑀦𑀻𑀬𑀚𑀸𑀢𑀓

(23.)

50. _“𑀅𑀧𑀺 𑀧𑀲𑁆𑀲𑁂𑀦 𑀲𑁂𑀫𑀸𑀦𑁄,_  
_𑀲𑀮𑁆𑀮𑁂𑀪𑀺 𑀲𑀮𑁆𑀮𑀮𑀻𑀓𑀢𑁄;_  
_𑀲𑁂𑀬𑁆𑀬𑁄𑀯 𑀯𑀍𑀅𑀯𑀸 𑀪𑁄𑀚𑁆𑀚𑁄,_  
_𑀬𑀼𑀜𑁆𑀚 𑀫𑀜𑁆𑀜𑁂𑀯 𑀲𑀸𑀭𑀣𑀻”𑀢𑀺𑁇_
