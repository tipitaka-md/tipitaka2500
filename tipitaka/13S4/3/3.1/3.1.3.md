# 𑁩𑁇𑁧𑁇𑁩 𑀆𑀯𑁂𑀡𑀺𑀓𑀤𑀼𑀓𑁆𑀔𑀲𑀼𑀢𑁆𑀢

### 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑀁𑀬𑀼𑀢𑁆𑀢

### 𑀧𑀞𑀫𑀧𑁂𑀬𑁆𑀬𑀸𑀮𑀯𑀕𑁆𑀕

### 𑀆𑀯𑁂𑀡𑀺𑀓𑀤𑀼𑀓𑁆𑀔𑀲𑀼𑀢𑁆𑀢

(282.)

1243\. “𑀧𑀜𑁆𑀘𑀺𑀫𑀸𑀦𑀺, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀆𑀯𑁂𑀡𑀺𑀓𑀸𑀦𑀺 𑀤𑀼𑀓𑁆𑀔𑀸𑀦𑀺, 𑀬𑀸𑀦𑀺 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀓𑀢𑀫𑀸𑀦𑀺 𑀧𑀜𑁆𑀘? 𑀇𑀥, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀤𑀳𑀭𑁄𑀯 𑀲𑀫𑀸𑀦𑁄 𑀧𑀢𑀺𑀓𑀼𑀮𑀁 𑀕𑀘𑁆𑀙𑀢𑀺, 𑀜𑀸𑀢𑀓𑁂𑀳𑀺 𑀯𑀺𑀦𑀸 𑀳𑁄𑀢𑀺𑁇 𑀇𑀤𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀧𑀞𑀫𑀁 𑀆𑀯𑁂𑀡𑀺𑀓𑀁 𑀤𑀼𑀓𑁆𑀔𑀁, 𑀬𑀁 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀧𑀼𑀦 𑀘𑀧𑀭𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀉𑀢𑀼𑀦𑀻 𑀳𑁄𑀢𑀺𑁇 𑀇𑀤𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀤𑀼𑀢𑀺𑀬𑀁 𑀆𑀯𑁂𑀡𑀺𑀓𑀁 𑀤𑀼𑀓𑁆𑀔𑀁, 𑀬𑀁 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀧𑀼𑀦 𑀘𑀧𑀭𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀕𑀩𑁆𑀪𑀺𑀦𑀻 𑀳𑁄𑀢𑀺𑁇 𑀇𑀤𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀢𑀢𑀺𑀬𑀁 𑀆𑀯𑁂𑀡𑀺𑀓𑀁 𑀤𑀼𑀓𑁆𑀔𑀁, 𑀬𑀁 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀧𑀼𑀦 𑀘𑀧𑀭𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀯𑀺𑀚𑀸𑀬𑀢𑀺𑁇 𑀇𑀤𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀘𑀢𑀼𑀢𑁆𑀣𑀁 𑀆𑀯𑁂𑀡𑀺𑀓𑀁 𑀤𑀼𑀓𑁆𑀔𑀁, 𑀬𑀁 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀧𑀼𑀦 𑀘𑀧𑀭𑀁, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀼𑀭𑀺𑀲𑀲𑁆𑀲 𑀧𑀸𑀭𑀺𑀘𑀭𑀺𑀬𑀁 𑀉𑀧𑁂𑀢𑀺𑁇 𑀇𑀤𑀁 𑀔𑁄, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀧𑀜𑁆𑀘𑀫𑀁 𑀆𑀯𑁂𑀡𑀺𑀓𑀁 𑀤𑀼𑀓𑁆𑀔𑀁, 𑀬𑀁 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀺𑁇 𑀇𑀫𑀸𑀦𑀺 𑀔𑁄, 𑀪𑀺𑀓𑁆𑀔𑀯𑁂, 𑀧𑀜𑁆𑀘 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑀲𑁆𑀲 𑀆𑀯𑁂𑀡𑀺𑀓𑀸𑀦𑀺 𑀤𑀼𑀓𑁆𑀔𑀸𑀦𑀺, 𑀬𑀸𑀦𑀺 𑀫𑀸𑀢𑀼𑀕𑀸𑀫𑁄 𑀧𑀘𑁆𑀘𑀦𑀼𑀪𑁄𑀢𑀺, 𑀅𑀜𑁆𑀜𑀢𑁆𑀭𑁂𑀯 𑀧𑀼𑀭𑀺𑀲𑁂𑀳𑀻”𑀢𑀺𑁇
