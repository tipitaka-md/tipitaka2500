# 3.3.7 Samattasutta

### Satipaṭṭhānasaṃyutta

### Sīlaṭṭhitivagga

### Samattasutta

(393.)

884. Taṃyeva nidānaṃ. Ekamantaṃ nisinno kho āyasmā sāriputto āyasmantaṃ anuruddhaṃ etadavoca—  “‘asekho, asekho’ti, āvuso anuruddha, vuccati. Kittāvatā nu kho, āvuso, asekho hotī”ti? “Catunnaṃ kho, āvuso, satipaṭṭhānānaṃ samattaṃ bhāvitattā asekho hoti.

885. Katamesaṃ catunnaṃ? Idhāvuso, bhikkhu kāye kāyānupassī viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ; vedanāsu…pe…  citte…pe…  dhammesu dhammānupassī viharati ātāpī sampajāno satimā, vineyya loke abhijjhādomanassaṃ. Imesaṃ kho, āvuso, catunnaṃ satipaṭṭhānānaṃ samattaṃ bhāvitattā asekho hotī”ti.

===

886. Sattamaṃ.
